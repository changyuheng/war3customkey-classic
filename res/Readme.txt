War3Customkey
-------------



功能：
───
1.血條常駐：就跟一直按著Alt的效果一樣。

2.鎖住左邊的 Windows Logo Key ：魔獸開啟時，左邊的 Windows Logo Key (視窗鍵)
  會被鎖住，不小心也按到不會跳回桌面。

3.修改物品欄快速鍵：
  將〔Tab〕、〔Caps Lock〕、〔Windows Logo Key〕對應第 1, 2, 3 個物品欄的熱鍵。
  註：〔〕引號內的是按鍵名稱，下面的說明中有用到按鍵名稱的部分，也會用引號將其括
  起來。

4.背景自動連線到 Battle.Net 。(「不會」嘗試錯誤的密碼登入而造成增加伺服器負擔。)



操作說明：
─────
1.按 〔[〕 鍵開啟友方單位血條常駐、按 〔]〕 鍵開啟敵方單位血條常駐。
  按 Ctrl + Q 終止血條常駐。

2.〔Tab〕鍵 = 〔Num 7〕
  〔Tab〕鍵對應九宮格數字區的 7 ，也就是按一下 〔`〕 就能第 1 格 (左上角)
  物品欄的物品。

3.〔Caps Lock〕 = 〔Num 8〕
  〔Caps Lock〕 鍵對應九宮格數字區的8，也就是按一下 〔Caps Lock〕 就能使用第 2
  格 (右上角) 物品欄的物品。

4.〔Windows Logo Key〕 = 〔Num 4〕
  〔Windows Logo Key〕鍵對應九宮格數字區的 4 ，也就是按一下〔Windows Logo Key〕
  就能使用第 3 格 (左邊中間) 物品欄的物品。

5.〔Ctrl〕+〔F11〕背景自動連線到 Battle.Net ，可在 Warcraft III 最小化時使用。
  〔F12〕終止自動連線。
